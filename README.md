# 📊 Retail Sales Exploratory Data Analysis (EDA)

## 🚀 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a retail sales dataset to uncover valuable business insights. The analysis includes data cleaning, preprocessing, visualization, and interpretation of customer purchasing behavior, sales trends, and product performance.

---

## 🎯 Objective

* Understand the retail sales data.
* Clean and preprocess the dataset.
* Identify sales trends and patterns.
* Analyze customer purchasing behavior.
* Generate business insights using visualizations.
